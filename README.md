## ✅ 참여 방법
1. 이 저장소를 `fork` 한다. 
2. `problem_solve`에 자동 생성된 데일리 문제 폴더에서 해당 문제 디렉터리를 확인한다.
3. 해당 문제 폴더에 자신의 소스코드를 업로드한다. 
4. 이때 `commit 규칙`을 지킨다.
5. 원본 저장소로 `Pull Request`를 보낸다. 
6. 다른 사람들의 PR을 보고 자유롭게 코드 리뷰를 한다.
